# Healthy Food Delivery App

A cross-platform mobile application for healthy food ordering and delivery, developed as a coursework project using React Native and Expo. 

## Tech Stack

*   **Framework:** React Native
*   **Platform:** Expo
*   **Language:** JavaScript (ES6+)

## Prerequisites

Before you begin, ensure you have the following installed:
*   **Node.js** (LTS version recommended)
*   **Expo Go** app installed on your physical mobile device (iOS or Android) or an Emulator set up on your computer.

## Getting Started

Follow these steps to get the application running locally.

### 1. Clone the Repository

```
git clone https://github.com/Bahdan-Mixajlov/healthy-food-delivery-mobile-app.git
cd healthy-food-delivery-mobile-app
```
### 2. Install Dependencies

Install the required node modules defined in package.json.

```
npm install
```


### 3. Run the Application

Start the Expo development server.

```
npx expo start
``` 
*Note: If you encounter caching issues, use npx expo start -c to start with a clear cache.*


Once the server is running, you will see a QR code in your terminal.

*   **On Android:** Scan the QR code using the Expo Go app.
*   **On iOS:** Open the Camera app, scan the code, and tap the notification to open in Expo Go.
